# 🧠 Evoastra Ventures Internship Assessment

**Phase 1 – Skill Evaluation (Data Analysis & Business Insights)**

## 📝 Project Overview

This repository contains my submission for the **Evoastra Ventures Internship Assessment – Phase 1**.

The objective of this assessment was to evaluate analytical thinking, problem-solving ability, data interpretation skills, and business communication.

Using an **E-commerce Customer Behavior dataset**, the analysis focuses on customer segmentation, profitability, churn prediction, and business strategy recommendations.

## 🏗️ Analysis Workflow

### 1️⃣ Data Understanding & Quality Assessment

* Identified data types and business relevance
* Checked for missing values and inconsistencies
* Evaluated churn distribution and return patterns

### 2️⃣ Customer Segmentation & Profitability

* Calculated net profit per customer segment
* Analyzed churn rates across product categories
* Evaluated customer lifetime value conceptually

### 3️⃣ Churn Prediction Approach

* Identified relevant features for classification
* Applied Logistic Regression as baseline model
* Evaluated using Accuracy, Precision, Recall, F1-score

### 4️⃣ Business Strategy & Recommendations

* Proposed retention strategies for high-risk customers
* Suggested marketing improvements for high-value segments
* Provided actionable insights to improve profitability

---

## 📊 Key Insights

* Electronics customers showed higher churn compared to Fashion customers.
* Returns had indirect impact on churn behavior when analyzed monthly.
* Middle-age segment generated strong profitability with moderate churn risk.
* Data-driven retention strategies can significantly improve long-term revenue.
